HOW CHESS SQUARES WORK — read this before examining the board
Each square has a letter (column a–h, left to right) and a number (row 1–8, White's side at bottom).
Column values: a=1   b=2   c=3   d=4   e=5   f=6   g=7   h=8

Pieces: K=King   Q=Queen   R=Rook   B=Bishop   N=Knight   (no letter = Pawn)
